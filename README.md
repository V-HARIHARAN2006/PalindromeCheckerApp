# PalindromeCheckerApp

## Project Title

**Palindrome Checker Application using Java**

---

## Project Description

PalindromeCheckerApp is a simple Java console application that checks whether the given word or number entered by the user is a palindrome or not.

A palindrome is a string that reads the same forward and backward.

Examples:

* MADAM → Palindrome
* LEVEL → Palindrome
* HELLO → Not a Palindrome

The program takes input from the user, reverses the string, and compares it with the original input to produce the result.

## Flow of the Project

1. Start the program
2. Ask the user to enter a word or number
3. Store the input using Scanner
4. Reverse the input string
5. Compare original string with reversed string
6. If both are equal → Palindrome
7. Else → Not a Palindrome
8. Display the result
9. End the program

---

## Working Procedure (Algorithm)

1. Begin
2. Read input from the user
3. Convert input into string
4. Reverse the string using loop
5. Compare reversed string with original string
6. If equal, print "Palindrome"
7. Else, print "Not a Palindrome"
8. Stop
